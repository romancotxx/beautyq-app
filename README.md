# БьютиQ

Telegram-бот (@beautyq_bot), который пишет посты, идеи и промпты для картинок в голосе конкретного beauty-салона. Владелец салона перестаёт думать о контенте: рассказал про месяц голосовым — получил план, нажал /post — получил готовый текст в своём стиле.

Стек: n8n Cloud (вся логика) + Google Sheets (база) + OpenAI (LLM/Whisper/Vision, целевая модель — Claude API) + Telegram Bot API + Mini App на GitHub Pages.

## Главные правила проекта

- Качество текстов — приоритет №1, фичи после
- Меньше кнопок и вопросов — лучше; бот не спрашивает то, что знает
- Оценка качества — только admin_score Романа, пользовательских лайков нет
- MVP над перфекционизмом
